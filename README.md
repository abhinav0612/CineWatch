Sample Movie app using Hilt, RxJava3, View Binding, Jetpack Components( View Model, Navigation Component, Room, Live Data),and Retrofit

### Cine Watch is demo app built on MVVM architecture.


## Libraries Used:
- Hilt
- RxJava3
- ViewModel
- LiveData
- ViewBinding
- Retrofit
- Room
- Android Youtube Video Player
- SmoothBottomBar

## API used:
